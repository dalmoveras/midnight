# Midnight

## A Deep Web Datamining and Search Tool

### Pre requisites
* python3
* virtualenv
* virtualenvwrapper
* tor

### Installing
1 - `mkvirtualenv midnight`
2 - `git clone https://github.com/murksombra/midnight`
3 - `cd midnight`
4 - `pip3 install requirements.txt`
